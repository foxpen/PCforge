# 🖥️ PCForge

Moderní PC konfigurátor pro CZ/SK trh. Porovnává ceny z 12+ e-shopů.

## Funkce
- ⚙️ Konfigurátor s kontrolou kompatibility
- ⚖️ Porovnání komponent
- 📦 Katalog s filtry
- 🎮 Can I Run It?
- 🏆 Achievement systém

## Lokální vývoj

```bash
npm install
npm run dev
```

## Deploy na Netlify

1. Push na GitHub
2. Připoj repo na [netlify.com](https://netlify.com)
3. Build command: `npm run build`
4. Publish dir: `dist`

Nebo automaticky přes `netlify.toml` v rootu.
